# Swapnil Patil - Developer Portfolio

A single-page, static, recruiter-facing developer portfolio website. Designed to be clean, fast, and project-first, optimized for quick scanner assessments.

## Technologies Used
- HTML5
- CSS3 (Vanilla CSS Custom Properties & Grid/Flex layout)
- JavaScript (Vanilla ES6, IntersectionObserver, and Fetch API)
- GitHub Pages (Deployment)
