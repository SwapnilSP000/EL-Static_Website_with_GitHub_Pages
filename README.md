# Developer Portfolio - GitHub Pages Static Website

This project is a clean, professional portfolio website for Swapnil Patil. It is built with pure HTML5, CSS3, and vanilla JavaScript and is ready to be hosted on GitHub Pages.

- Responsive design for desktop, tablet, and mobile
- Clean and professional UI
- GitHub Pages ready structure
- Mobile-friendly navigation
- Pure HTML, CSS, and JavaScript
- Fast loading and lightweight performance
- SEO friendly metadata and semantic HTML

## Installation

1. Clone the repository.
2. Open the project folder in your editor.
3. Open index.html directly in a browser, or use VS Code Live Server.

## GitHub Pages Deployment

1. Push the project to GitHub.
2. Open the repository on GitHub.
3. Go to Settings.
4. Select Pages from the sidebar.
5. Choose Deploy from a branch.
6. Select the main branch and root folder.
7. Save the settings.
8. Wait for deployment to complete.
9. Open the published website URL.

## Customization

You can update the following sections easily:

- Profile details in index.html
- Projects and descriptions in the Projects section
- Skills cards in the Skills section
- Experience cards in the Experience section
- Social links and contact information in the Contact section

- Images in assets/images
